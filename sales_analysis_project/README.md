# Sales Analysis Project

Projeto para análise de vendas com base em transações, produtos e lojas.

## Pastas

- `data/` — arquivos CSV de entrada.
- `sql_queries/` — consultas para modelagem e KPIs.
- `notebooks/` — análises exploratórias.
- `dashboard/` — documentação para visualização.

## Arquivos de dados

- `sales_transactions.csv`
- `products.csv`
- `stores.csv`

## Queries disponíveis

- `create_sales_tables.sql`
- `sales_kpis.sql`
- `monthly_revenue.sql`

## Opcional: usar Kaggle

Você pode substituir os CSVs da pasta `data/` por datasets do Kaggle mantendo os mesmos nomes de colunas.
