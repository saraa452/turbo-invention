# Business Dashboard

Projeto para criação de dashboards executivos usando dados de KPI e performance de canais.

## Pastas

- `datasets/` — bases de indicadores
- `powerbi/` — arquivos e documentação do painel

## Dados disponíveis

- `business_kpis.csv`
- `channel_performance.csv`

## Opcional: usar Kaggle

Você pode trocar as bases por datasets públicos de marketing/performance e manter o modelo de colunas para reaproveitar visuais.
